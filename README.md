# Fansten Landing

Standalone marketing website workspace for the Fansten landing page.

## Local run
```powershell
cd C:\Users\RichComputer\Desktop\fansten-site
npm.cmd run dev
```

## Checks
```powershell
npm.cmd run typecheck
npm.cmd run build
```

## Notes
- this workspace is isolated from the Fansten mobile app and admin panel
- the landing uses the approved hero asset from `public/images/fansten-hero.png`
- future routes like `/ru`, `/app`, and `/admin` are intentionally not implemented here yet
